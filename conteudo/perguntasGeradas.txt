Questão 5

5 - [MODIFICACAO] Como modificar o código para que ele também valide se o número é palíndromo ignorando os espaços e caracteres não numéricos?
 
 
 
 
Resposta 5 -
 
import re

texto = input("Digite algo: ")

# remove tudo que não é número
numero = re.sub(r'[^0-9]', '', texto)

# verifica palíndromo
if numero == numero[::-1]:
    print("É palíndromo")
else:
    print("Não é palíndromo")
