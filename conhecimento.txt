QUESTOES :


1- Contagem de vogais em uma string, Crie um programa que receba uma string e conte quantas vogais existem nela.
2 - Números primos em um intervalo. Faça um programa que exiba todos os números primos entre 1 e 50.
3- Soma dos elementos de uma matriz. Dada uma matriz 3x3, calcule a soma de todos os elementos.
4 -Inverter uma string sem usar slicing .Implemente um programa que inverta uma string sem usar [::-1].
5 - Verificar número palíndromo. Crie um programa que verifique se um número é palíndromo (ex: 121, 1331).








Resposta 1 -

texto = input("Digite uma frase: ").lower()
vogais = "aeiou"
contador = 0

for letra in texto:
    if letra in vogais:
        contador += 1

print("Quantidade de vogais:", contador)





Resposta 2 -

for num in range(2, 51):
    primo = True

    for i in range(2, num):
        if num % i == 0:
            primo = False
            break

    if primo:
        print(num)



Resposta 3 -

matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

soma = 0

for linha in matriz:
    for elemento in linha:
        soma += elemento

print("Soma total:", soma)




Resposta 4 - 

texto = input("Digite uma palavra: ")
invertida = ""

for i in range(len(texto)-1, -1, -1):
    invertida += texto[i]

print("Invertida:", invertida)






Resposta 5 -
numero = input("Digite um número: ")

invertido = numero[::-1]

if numero == invertido:
    print("É palíndromo")
else:
    print("Não é palíndromo")
